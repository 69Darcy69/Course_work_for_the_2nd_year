Курсовая работа за 2 год обучения.
Техническое задание
Разработать программу, демонстрирующую применение внешних интерфейсов взаимодействия с операционной системой. Требования к программе:
–	Программа работает в графическом режиме.
–	Программа использует кнопки для ввода данных (операций) и графическое меню.
–	Программа содержит поле для ввода данных и вывода результата.
–	Вывод календаря.
–	Возможность выбора даты в календаре.
–	Возможность создания события на выбранную дату.
–	Возможность вывода события с выбранной даты.
